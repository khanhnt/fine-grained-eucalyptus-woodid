# Results Summary

These CSVs store the cleaned final Split B benchmark results used in the manuscript.

- `test_results_splitB_4models_3seeds.csv`: one row per model/seed.
- `splitB_final_multiseed_summary.csv`: mean and standard deviation across the three seeds.

The final table uses the fresh rerun values for affected seed-3407 ResNet-50 and ViT-B/16 runs.
